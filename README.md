# 🐺 FocusWolf

Aplicación móvil desarrollada en Android Studio con Kotlin que ayuda a estudiantes a organizar sus tareas académicas mediante proyectos, prioridades y fechas límite.

---

## 📌 Descripción

FocusWolf es una app de productividad académica que permite a los estudiantes gestionar sus tareas de forma clara y estructurada.

La aplicación permite:

- Crear tareas
- Agrupar tareas por proyectos
- Asignar prioridades
- Establecer fechas límite
- Dar seguimiento al progreso
- Visualizar estadísticas

Su objetivo es reducir la sobrecarga académica y mejorar la gestión del tiempo.

---

## 🎯 Público Objetivo

- Estudiantes universitarios  
- Estudiantes de preparatoria  
- Personas que manejan múltiples tareas académicas
